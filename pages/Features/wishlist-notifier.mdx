# Wishlist Notifier

![Wishlist Notifier](/img//wish2.png)

## How to use
<Steps>
### Enabling this feature
First, if you want to use this feature, you need to enable it in your server. How? If you are an admin, just run the `[p]wishwatch` command and then click the <button className="btn btn-success">Enable this feature</button> button

![Wishlist Notifier](/img/wish3.png)

### Command
After that you can run the command below
- `[p]wishwatch`

![Wishlist Notifier](/img/wish.png)

### Removing my Wishlist Ping Role
- To remove your <span className="mention">@Wishlist Ping</span> role, just click the <button className="btn btn-danger">Disable My Ping</button> button
</Steps>
Note : You can use this command on any servers

<Callout type="warning">
<b className="mention">@Keqing</b> must have the <code>MANAGE ROLES</code> perms
</Callout>

<Callout type="info">
The role can be renamed as long as it includes Wishlist anywhere, case-insensitive
</Callout>

## Role restriction
<Callout type="warning">
Only for premium server
</Callout>
For premium servers, you can set the `Role requirement` to run the `[p]wishwatch` command. Means you can combine it with `mee6` or other bots. 

Format : 
- [p]wishwatch &lt;<span className="mention">@role</span> | false>
- Use **false** to make it available to all members.

## Disable Feature
- Run `[p]dropwath` command
- Press the <button className="btn btn-danger">Disable this feature</button> button

<Callout type="warning">
Important <span className="mention">@Keqing</span> will delete the <span className="mention">@Wishlist Ping</span> role, which means all members will lose the role.
</Callout>
