# Sketch Gallery

<Callout emoji="⚠️">
This feature is available for **premium servers** only
</Callout>

![Sketch Gallery](/img/sketch.png)

### How to use

The `[p]sketch` feature will display any known sketches of a specific character or series, sorted by the amount of ink used in the sketch. To activate, simply run this command

### Command
- [p]sketch [filter]

### Available filters
- c: characterName
- s: seriesName
- o: ownerId
- sk: sketcherId

### Example
- `asketch c:keqing s:genshin impact`
