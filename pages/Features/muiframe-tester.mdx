# Multiframe Tester

<Callout emoji="⚠️">
This feature is available for **premium servers** only
</Callout>

![Multiframe tester](/img/getframes.gif ':size=100%')

## Command Information

### Command
<Callout type="info">
Format: `[p]getframes <type> [page]`
  - `<type>` = bits, carousel, special, event, transparent, monthly, ranked, bits type (e.g wood, stone), or Frame names
  - `[page]` = optional
</Callout>

### Aliases
- `[p]getf`
- `[p]getframe`
- `[p]multif`
- `[p]multiframe`

### Example
```bash
agetframes bits 1
amultiframe carousel 3
agetf special 5
amultif event 2
agetf stone 1
agetframes budokai, arcana, interface, petals
```

### Usage

<Steps>
### Search for the character through `k!lookup`
Example: `k!lookup sinon`
### Select the corresponding edition
Click the ⬅️ or ➡️ to switch edition
### Reply to the lookup embed with the multi-frame tester command
Example: `agetframes transparent`
</Steps>
