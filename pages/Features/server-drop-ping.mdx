# Server Drop Ping

![](/img/dropwatch.png ':size=100%')

![Server Drop](/img/drop.png ':size=100%')

Basically same as the [Wishlist Notifier](./wishlist-notifier)

## How to use
<Steps>
### Enabling this feature
First, if you want to use this feature, you need to enable it in your server. How? If you are an admin, just run the `[p]dropwatch` command and then click the <button className="btn btn-success">Enable this feature</button> button

![Server Drop](/img/drop2.png ':size=100%')

### Command
After that you can run the command below
- `[p]dropwatch`

### Removing my Server Drop Ping Role
- To remove your <span className="mention">@Server Drop Ping</span> role, just click the <button className="btn btn-danger">Disable My Ping</button> button
</Steps>
Note : You can use this command on any servers

<Callout type="warning">
<b className="mention">@Keqing</b> must have the <code>MANAGE ROLES</code> perms
</Callout>

## Role restriction

This feature can be limited so that a role is required to obtain the <span className="mention">@Server Drop Ping</span>

Format : 
- [p]adropaccess &lt;<span className="mention">@role</span> | false>
- Use **false** to make it available to all members.

## Disable Feature
- Run `[p]dropwath` command
- Press the <button className="btn btn-danger">Disable this feature</button> button

<Callout type="warning">
Important <span className="mention">@Keqing</span> will delete the <span className="mention">@Server Drop Ping</span> role, which means all members will lose the role.
</Callout>
