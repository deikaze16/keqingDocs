### How to use
- First, `klu` a character
- Select it's Edition
- `Reply` to the embed of `klu` command

![](/img/features/reply.png)

- Run the command below


:::info Command:
**Usage** : `[p]getframes  <frame type> [page]`

**Aliases** : `[p]getf`, `[p]getframe`, `[p]multif`, `[p]multiframe`

**Example**  :
```bash
agetframes bits 1
amultiframe carousel 3
agetf special 5
amultif event 2
```
:::
